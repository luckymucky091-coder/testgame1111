# Quiz Land P6

A pixel-art coin-collecting jumping game for 6th-grade computer science practice. Players collect coins that trigger MCQ questions; incorrect answers lose hearts. Built with HTML/CSS/JS for browser & mobile.

Features
- Landscape gameplay, pixel art UI
- Random 10-question round from a bank of questions
- Four hearts (lives); lose a heart per wrong answer
- Score by collecting all coins and reaching the school
- Music & SFX with mute/unmute controls
- Playable on Android, iOS, and PC via the browser

How to run
1. Open `index.html` in a modern browser.
2. For local testing on mobile, serve with a simple static server (e.g., `npx http-server`).

Project structure
- index.html — entry
- style.css — basic styling
- game.js — game logic
- assets/ — images, audio
- data/questions.json — question bank
